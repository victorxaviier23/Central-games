# 🎮 Central-games

## Sobre o Projeto

**Central-games** é um projeto destinado à disciplina de **Banco de Dados**, focado no desenvolvimento de um **Sistema de Gerenciamento de uma Lan House**.

Este sistema visa facilitar a gestão de operações, clientes, equipamentos, aluguéis de máquinas e demais funcionalidades essenciais para uma lan house moderna.
